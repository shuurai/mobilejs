Documentation

Latest Version: 2.1
This documentation should explain the foundamentals, design, development to the deployment of the DG Web App. Although the version in the current phase should support responsiveness but snippets for `large` and `medium` screens have been disabled or removed.

Developers and designers are required to have basic understanding of SCSS, but at least a intermediate level of JS and mobile front grid concepts.

The design of this template is done is such a way to allow easy integration of the web app into a PhoneGap wrapper or alike. It should also be easily deployed on to CDN with no attachment to any backend or dynamic generations to allow cached and faster access.
